Deep Learning for Wireless Community Networks

Using deep learning to classify packets coming to and from a community network to ensure QoS in a low resource environment.

This project was done by computer science honours students Boitumelo Mokoka, Michael Gamsu and Ryan Murphy.

Context and Goals of this Project

The goal of this project is to classify network packets into their respective classes. The classes are the application they are associated with e.g classifying a Youtube packet as Youtube. This classifier did not only need to be accurate but also fast and memory efficient in order for a community network to use it.

Community Networks

Community networks are community governed and operated network infrastructures for digital communication. The users are the ones who run them and manage them with funding coming from the users themselves. They are extremely important in low resource communities as they provide low-cost internet access.
